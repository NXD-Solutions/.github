# Component Declaration

When a bounded component has principals, it declares its context in a `CLAUDE.md` at its root. Each principal authors their own H1 section, named after their role, containing their declaration for the component.

No section is added if the principal has nothing to declare. No file is needed if no principals have anything to declare.

**Example:** `# Charter` is the commissioning principal's section.

## Placement

`CLAUDE.md` lives at the root of the scope it governs — not above it, not below it. Move the scope and the file moves with it.
