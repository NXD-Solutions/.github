# Testament

Maintain `.claude/rules/claude-code/testament-log.md` — a local log of recurring problems specific to this repo. It lives only in `.claude/` and is never added to `.claude-static/`. Permanent fixes go through a normal PR.

**Add an entry** when the user hints a problem was solved before, or proactively when a non-obvious workaround is discovered.
**Remove an entry** when permanently fixed (e.g. promoted to CLAUDE.md).

Entry format:
```
## <Title>
**Symptom:** ...
**Solution:** ...
**Permanent fix candidate:** ...
```
