# Promote to Static

`.claude-static/` is the source of truth — changes start there. `.claude/` is the local working copy, derived from `.claude-static/`.

## How to make a change

1. Edit `.claude-static/<subdir>/<file>.md` — the canonical change
2. Derive the local copy:

```bash
cp .claude-static/<subdir>/<file>.md .claude/<subdir>/<file>.md
```

3. Stage and commit both in the same branch. Both ship in the same PR.

## Scope

Every `.md` file in `.claude-static/` must have a corresponding copy in `.claude/`. Exceptions:
- `.local.md` — local-only, lives in `.claude/` only, never in `.claude-static/`
- `.gen.md` — autogenerated, derive via generation script, not manually
