# thinking-core/

Portable, timeless thinking principles — independent of NXD, software development, and AI tooling.

## Placement gate

A file belongs here only when all three invariants hold simultaneously:

1. **Universal** — no dependency on NXD, software development, or AI tooling
2. **Timeless** — the principle survives context change; it does not age with a tool or platform
3. **No domain dependency** — no named tools, platforms, or org terms appear as load-bearing references

Failing any one invariant places the file in `nxd-practices/` instead.
