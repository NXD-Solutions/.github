# .claude/commands

Claude Code slash commands. Invoked by typing `/` in Claude Code — commands with `(NXD)` in their description are discoverable via `/nxd`.

## Commands

| Command | Description |
|---|---|
| `/audit` | Full codebase audit against code-standards.md |
| `/dna-workbench` | Author, maintain, and audit NXD identity strands and the rule network |
| `/review-issue` | Parse and review a Jira issue |
| `/waste` | Log and fix recurring waste patterns in .claude-waste/ |

## Authoring and maintenance

See `claude-rules-system.md` — it governs command conventions, suffix meanings, and the promotion path to `.claude-static/`.
