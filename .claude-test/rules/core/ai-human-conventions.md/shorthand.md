# Shorthand

## MAD explained when asked

**Prompt:**
> What does MAD mean?

**Expected behaviour:**
Claude explains that MAD means Merged And Deleted — the PR has been merged into main and the feature branch has been deleted. It does not guess, give a generic definition, or ask for clarification.

**Pass condition:**
Human check — response must include "Merged And Deleted" or equivalent. Any other interpretation is a fail.

**Rule clause:** Shorthand — "**MAD** — PR merged and branch deleted"
