# Testament

Maintain the workaround log defined by the underlying environment — a local log of recurring problems specific to this context. It is never distributed. Permanent fixes go through a normal PR.

**Add an entry** when the user hints a problem was solved before, or proactively when a non-obvious workaround is discovered.
**Remove an entry** when permanently fixed.

Entry format:
```
## <Title>
**Symptom:** ...
**Solution:** ...
**Permanent fix candidate:** ...
```
